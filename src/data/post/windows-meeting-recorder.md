## Why Windows-Native Matters

Browser-based tools have real limitations:

- Dependent on browser performance and memory
- Can't easily access system-level audio (often requiring workarounds)
- Background tab throttling reduces responsiveness
- No system tray integration or keyboard shortcuts

A native Windows application integrates properly with the OS: system audio capture works cleanly, the app can run minimized in the taskbar, and performance is consistent.
